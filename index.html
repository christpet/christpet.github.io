Entreprendre et Coder

A (not quite) free, responsive way to learn how to code your landing page.

On commence

Mission du jour

Aujourd'hui, nous allons apprendre comment héberger gratuitement sur GitHub (enfin !) et connecter un formulaire d'inscription avec MailChimp.

Première étape

La ligne de commande (the command line) et Git. Aller ici pour commencer. Trouvez la "terminal" ou la "command line" sur ton ordinateur, et suivez les étapes.

Deuxième étape

Après avoir vu "hello world" lorsque vous naviguez vers "[votre nom d'utilisateur].github.io" dans votre navigateur, copiez les dossiers de votre site Web dans le dossier "[votre nom d'utilisateur].github.io" de votre ordinateur. Avertissement : Vous pourriez aussi avoir besoin de changer les tags du script CSS dans votre fichier HTML !

Troisième étape

[Mode Hard] Pour mettre un formulaire d'inscription à votre site Web, créez d'abord un compte MailChimp, puis suivez les instructions à ce lien. Pour un formulaire d'inscription un peu plus facile à intégrer, allez sur ce lien.
